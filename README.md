# Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project uses **K-Means Clustering**, an unsupervised machine learning algorithm, to segment customers based on their behavior and demographic characteristics. The project covers the complete data analysis workflow, from data preprocessing to business insights.

---

## 🎯 Objectives
- Clean and preprocess customer data
- Perform Exploratory Data Analysis (EDA)
- Select important features
- Scale features for clustering
- Determine the optimal number of clusters using the Elbow Method
- Apply K-Means Clustering
- Visualize customer segments
- Generate business insights and recommendations

---

10. K-Means Clustering
11. Cluster Visualization

13. Business Insights
14. Conclusion

---

## 📊 Key Results

- Successfully segmented customers into **4 clusters**
- Identified loyal, premium, new, and high-risk customer groups
- Analyzed customer behavior using tenure, monthly charges, and total charges
- Generated actionable business recommendations for customer retention and targeted marketing
