# Customer-Segmentation-Project
A Customer Segmentation project built using Python and K-Means Clustering to group customers based on their behavior. This project includes data cleaning, exploratory data analysis (EDA), feature engineering, feature scaling, clustering, visualization, and business insights using the Telco Customer Churn dataset.
# Customer Segmentation using K-Means Clustering

## Project Overview
This project applies K-Means Clustering to segment customers based on their behavior and demographic information. The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, feature scaling, clustering, visualization, and business insights.

## Objectives
- Clean and preprocess customer data
- Perform Exploratory Data Analysis (EDA)
- Select and scale important features
- Apply K-Means Clustering
- Analyze customer segments
- Generate business insights

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Workflow
1. Import Libraries
2. Load Dataset
3. Data Understanding
4. Data Cleaning
5. Exploratory Data Analysis
6. Feature Engineering
7. Feature Selection
8. Feature Scaling
9. Elbow Method
10. K-Means Clustering
11. Cluster Visualization
12. Customer Segment Analysis
13. Business Insights
14. Conclusion

## Key Insights
- Identified four distinct customer segments.
- High-value customers showed long tenure and higher spending.
- New customers had a higher risk of churn.
- The analysis provides insights for targeted marketing and customer retention.

Telco Customer Churn Dataset
